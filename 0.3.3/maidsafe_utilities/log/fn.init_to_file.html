pub fn init_to_file<P: AsRef<Path>>(show_thread_name: bool, file_path: P) -> Result<(), String>

Initialises the env_logger for output to a file and to stderr.

This function will create the logfile at file_path if it does not exist, and will truncate it if it does. For further details, see the module docs.
